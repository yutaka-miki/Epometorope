# エポメトロープとは？
エポメトロープとは、デジタルファブリケーションやオープンソースハードウェア、市販品を組み合わせて作ることができる全周観察型立体ディスプレイ（360度どの角度からでも観察することのできる立体像を表示できる装置）です。
